補助金・加算は、申請から「結果」まで責任を持ちます

制度の説明だけでは終わりません。書類準備、スケジュール管理、要件適合性チェック、申請後のフォローまで対応。生産性向上推進体制加算など複雑な要件の制度でも、施設の経営計画と紐付けて伴走します。

補助金活用／加算取得
D

機器導入後は、現場に通って職員研修と運用調整を実施。夜勤帯の使い方、記録のフロー、トラブル時の対応まで、現場の言葉で一つずつ整理します。「導入したのに使われない」を、私たちは経験させません。

現場定着／運用最適化
E

60を超える商材から、本当に必要なものだけ選びます

商材中立／最適選定

— FAQ

よくあるご質問

ご相談前によくいただくご質問をまとめました。ここにない疑問も、お気軽にお問い合わせください。

Q 「DXに何をすればいいか分からない」段階でも、相談していいですか? +
はい、もちろんです。むしろ「何をすればいいか分からない」段階こそ、私たちにご相談ください。「DXしたい」ではなく「何に困っているか」から伺います。資料の準備や事前の検討は一切不要です。30分の無料相談で、まずは方向性を一緒に整理します。
Q 補助金は本当に取れるんですか? +
施設の規模、業種、現在の体制、地域によって、活用できる制度は大きく異なります。生産性向上推進体制加算、ICT導入補助金、地域医療介護総合確保基金など、施設に最適な制度を抽出し、要件適合性をチェックした上で申請をサポートします。すべての施設で必ず取れるとは申し上げられませんが、活用余地は必ずお伝えします。
Q 小規模な施設でも対応してもらえますか? +
はい、対応可能です。グループホーム、サービス付き高齢者向け住宅、デイサービスなど、規模を問わずご相談ください。小規模だからこそ、機器投資の判断が経営に直結します。施設規模・予算に応じて、段階的な導入プランをご提案します。
Q 現場の職員から反発が出ないか心配です。 +
多くの施設様が同じ不安を抱えていらっしゃいます。私たちは、機器の選定段階から現場リーダーに参加いただき、「現場の声から始まる」プロセスを大切にしています。導入時は職員研修・運用調整を丁寧に行い、夜勤帯やシフト変更などの実情に合わせて運用を組み上げます。「使いこなせない」を残さないことが、私たちの最大のお約束です。
Q 既に他社の機器・ソフトを導入済みですが、連携できますか? +
可能な限り既存資産を活かす方向でご提案します。介護ソフト、見守り機器、ナースコール、Wi-Fi環境などはメーカー間の連携可否を確認した上で、必要に応じて中継ツールや運用設計でカバーします。すべてを入れ替える必要はありません。
Q 費用感はどれくらいですか? +
選定する機器・サービス、施設規模、補助金活用の有無によって大きく異なります。たとえば見守りシステム単独の導入と、施設全体のDX設計では数十倍の幅が出ます。診断・プラン提示の段階で、複数パターンの概算費用とROI試算をお示しします。費用面で無理のない範囲から始められる構成も、必ず一案ご提示します。
Q 導入後のサポート期間は決まっていますか? +
基本のサポート期間は導入から1年間ですが、契約形態に応じて月次レビュー型の継続フォロー、年次の制度改定対応など、長期パートナーとしての関わり方をご用意しています。「導入したら終わり」ではなく、施設の変化にあわせて長くお付き合いするのが私たちのスタンスです。
Q 個人情報や入居者データの取り扱いは大丈夫ですか? +
介護記録・健康情報・入居者個人情報など、すべて個人情報保護法および関連ガイドラインに準拠して取り扱います。データを取り扱うシステムについては、適切なセキュリティ要件を満たした製品のみを採用し、運用面でも職員アクセス権限の設計までサポートします。
